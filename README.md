# Git Mukashibanashi

このリポジトリは、GitHubの使い方を練習するためのサンドボックスです。
日本の昔話のテキストファイルを編集しながら、Gitの操作を学びます。

## 使い方

1. このリポジトリをForkしてください。
2. 自分のローカル環境にCloneしてください。
3. `stories` フォルダ内の昔話ファイルを自由に編集したり、新しい物語を追加したりしてください。
4. 変更をCommitし、Pushしてください。
5. Pull Requestを作成する練習をしてみましょう。

## ディレクトリ構成

- `stories/`: 昔話のテキストファイルが格納されています。
- `CONTRIBUTING.md`: コントリビュート（貢献）の手順について説明しています。
- `.gitignore`: GitHubにアップロードしないファイルを指定する設定ファイルです。

## セキュリティについて

パスワードやAPIキーなどの機密情報をGitHubにアップロード（公開）してはいけません。
`.env` ファイルなどの機密情報は、`.gitignore` ファイルに記述することでGitの管理対象から外すことができます。
本リポジトリでは練習用に `.gitignore` を配置しています。
