# コントリビュートガイド

このプロジェクトへの貢献（練習）手順です。

1. **Fork**
   このリポジトリを自分のアカウントにForkします。

2. **Clone**
   ForkしたリポジトリをローカルにCloneします。
   ```bash
   git clone https://github.com/YOUR_USERNAME/git-mukashibanashi.git
   ```

3. **Branch**
   新しいブランチを作成して作業します。
   ```bash
   git checkout -b fix/momotaro-typo
   ```

4. **Change**
   ファイルを編集します。

5. **Commit**
   変更をコミットします。
   ```bash
   git add .
   git commit -m "桃太郎の誤字を修正"
   ```

6. **Push**
   自分のリモートリポジトリにPushします。
   ```bash
   git push origin fix/momotaro-typo
   ```

7. **Pull Request**
   GitHub上でPull Requestを作成します。
